# CS 300 Module Eight Journal

## Reflection

The main problem I was solving in this course was figuring out how to store, organize, search, and display course information efficiently. Throughout the projects, I compared different data structures like vectors, linked lists, binary search trees, and hash tables to understand which one worked best for different situations. In Project Two, I created a program that could load course data from a file, sort the courses in alphanumeric order, and display course information, including prerequisites.

I approached each assignment by first understanding what the program needed to accomplish before deciding which data structure would work best. Learning about different data structures showed me that there is not one solution that works for every problem. Some structures are better for searching, while others are better for sorting or storing large amounts of data. Understanding these differences helped me make better decisions while designing programs.

One of the biggest roadblocks I faced was making sure my program correctly loaded the course file and handled prerequisites without crashing when there was missing or incorrect data. I worked through these problems by breaking the program into smaller parts, testing each function individually, and fixing errors one step at a time. Writing pseudocode before coding also helped me organize my thoughts and avoid mistakes.

This project expanded the way I think about designing software because I now spend more time planning before I start writing code. Instead of jumping straight into programming, I think about the algorithms, the data structure, and how each function will work together. That planning made my programs easier to build and debug.

My work in this course also changed the way I write code. I learned to separate tasks into smaller functions, use meaningful variable names, and keep my code organized so it is easier to read and update later. I also understand the importance of writing code that can be expanded if new features need to be added in the future. Overall, this course gave me a much better understanding of data structures, algorithms, and how they work together to solve real programming problems.
